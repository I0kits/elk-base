#ELK base

The repo aim to provide a template help you create a ELK cluster quickly.

### Tools
* [Docker](http://docker.io/)
* [Docker compose](https://github.com/docker/compose/releases)

### Instructions
* Start/Stop cluster
```
  docker-compose up
  docker-compose down
```
